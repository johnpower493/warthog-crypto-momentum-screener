Windows quickstart:

Backend
1) Create venv and install deps
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r backend\requirements.txt

2) Run backend from repo root (note --app-dir)
   python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload --app-dir backend

3) Check health
   curl http://localhost:8000/health

4) Check symbols discovery
   curl http://localhost:8000/debug/symbols
   # Expect a list of ~30 symbols (configurable)

Frontend
1) cd frontend
2) npm install
3) $env:NEXT_PUBLIC_BACKEND_WS="ws://localhost:8000/ws/screener"  # PowerShell
4) npm run dev

Troubleshooting
- If symbols count is 0: network issue or Binance REST blocked
- If symbols ok but no snapshots: wait up to 60s for first closed 1m candle; watch backend logs
- Ensure Python >= 3.9
- Check firewall allows outbound to fstream.binance.com (wss) and fapi.binance.com (https)
